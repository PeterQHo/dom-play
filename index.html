The goal of the project is when an actor is clicked on the page will highlight all of the actor lines on the page. This is done by putting all of the p tag elements in a div and to put the text in a span tag to highlight the words and not the p tag line on the page. By creating a id of "play" for the div I was able to select that id. I create a let function of "span" assigning it the value of document.querySelectorAll("#play span"); to select the id of the div and to select all the descendant span tags in it. I used the data attribrute and called in data-actor="Name of the actor" in my span tag so Java can tell the actors apart. Next I built an anonymous function into our event listener loop and than created a new function named highlightActor(mySpan.dataset.actor), in which we pass in the identity of the current actor to the function so that when clicked, the element always knows its name. Then using a conditional we say that if actor == mySpan.dataset.actor than the background-color of all the actor line will height yellow and other actors will be white.
